# CRM24 Premium Detail Checklist

## Главное

- [ ] Сначала создан `/ui-preview` или dev-only preview-экран.
- [ ] Рабочие страницы CRM не переписаны вслепую до принятия preview.
- [ ] Внешний вид заметно отличается от старого интерфейса.
- [ ] Это не старая CRM с перекрашенными карточками.

## Микродетали

- [ ] У каждого input есть label, hint, focus, error, disabled.
- [ ] Все input читаемые в светлой и тёмной теме.
- [ ] Кнопки имеют default, hover, pressed, focus, disabled.
- [ ] Badges выглядят как часть системы, а не как цветные пятна.
- [ ] Карточки не растянуты бессмысленно на wide-экранах.
- [ ] Sidebar не выглядит пустым.
- [ ] Topbar не налезает на content.
- [ ] Drawer/backdrop под цвет темы, не серый грязный слой.
- [ ] Меню «Ещё» выглядит как нормальный bottom sheet.

## Responsive

- [ ] 360x800 без горизонтального скролла.
- [ ] 390x844 без горизонтального скролла.
- [ ] 768x1024 не ломается.
- [ ] 1280x800 не ломается.
- [ ] 1440x900 выглядит полноценно.
- [ ] 1920x1080 использует пространство.
- [ ] 2560x1440 не выглядит как пустая простыня.

## Лиды и статистика

- [ ] Лиды на узких экранах карточками, без x-scroll.
- [ ] Поиск и фильтры выглядят премиально.
- [ ] Действия лида сохранены.
- [ ] Комментарии сохранены.
- [ ] Ответственный и статусы сохранены.
- [ ] Drill-down статистики сохранён: сотрудник -> стадия -> конкретные лиды.

## Чат

- [ ] Чат имеет список диалогов.
- [ ] На wide-экране чат может быть dock справа.
- [ ] На узких экранах чат обычный раздел.
- [ ] Bubbles аккуратные и привычные.
- [ ] Voice waveform красивый.
- [ ] Video circle виден при записи.
- [ ] При просмотре video circle увеличивается.
- [ ] Recording circle вылетает из composer.
- [ ] Camera loading имеет shimmer/ring.
- [ ] Attachments, replies, timestamps не выглядят случайно.

## Почта и поддержка

- [ ] Почта выглядит как привычный mail-клиент.
- [ ] Есть папки, список писем, reader, composer.
- [ ] Support inbox показывает обращения как диалоги.
- [ ] При клике на обращение открывается чат.
- [ ] Справа карточка компании/пользователя.

## Анимации

- [ ] Нет `transition: all`.
- [ ] Overlay lifecycle синхронный.
- [ ] Backdrop и panel стартуют вместе.
- [ ] Backdrop не исчезает раньше panel.
- [ ] Большие блоки не анимируют width/height/top/left/margin/padding.
- [ ] Reduced motion работает.
- [ ] Нет лагов и дерганий.

## Сдача

- [ ] Приложены screenshots всех ключевых экранов.
- [ ] Приложены screenshots 360, 390, 768, 1280, 1440, 1920, 2560.
- [ ] Приложено короткое видео/trace по drawer, chat, video circle.
- [ ] Build проходит.
- [ ] Нет React warnings и console errors.
