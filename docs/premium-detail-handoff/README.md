# CRM24 Premium Detail Handoff

Пакет для Codex, когда скачивание ZIP с телефона не работает.

Главный смысл: сначала нужно сделать отдельный `/ui-preview` с новым визуалом и микродеталями, а не сразу перекрашивать рабочую CRM.

Файлы:

- `docs/premium-detail-handoff/CODEX_PROMPT.md` — основной промт для Codex.
- `docs/premium-detail-handoff/CHECKLIST.md` — чеклист приёмки.

Правило: если после работы интерфейс визуально похож на старый с другими цветами — задача не выполнена.

Авторизацию, API, бизнес-логику, статусы, роли и рабочие сценарии не менять без отдельного разрешения.
